# WebApp Demo Project

A modern web application demo showcasing HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Works on all screen sizes
- **Smooth Animations** - CSS transitions and keyframe animations
- **Interactive Elements** - JavaScript-powered interactivity
- **Dark Theme** - Modern dark color scheme

## Files

- `index.html` - Main HTML structure
- `styles.css` - CSS styling with variables
- `app.js` - JavaScript functionality

## Getting Started

1. Open `index.html` in a browser
2. Or use the IDE preview panel

## Editing

Try editing:
- Change colors in `styles.css` (look for `:root` variables)
- Modify text in `index.html`
- Add new features in `app.js`

## Learn More

This project demonstrates:
- CSS Grid and Flexbox layouts
- CSS custom properties (variables)
- Intersection Observer API
- Event handling
- DOM manipulation
